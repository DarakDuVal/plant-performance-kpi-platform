# Database

Database for the plant performance KPI platform. This directory will hold the schema, migrations, and seed data used by
the [`../backend/`](../backend/) service.

> **Status: skeleton.** No schema yet. The database engine and migration tooling will be chosen as requirements firm up
> and recorded as an [ADR](../docs/decisions/).

## Intended contents

- Schema definitions for plant performance KPI data.
- Versioned migrations.
- Seed and fixture data for local development and tests.
